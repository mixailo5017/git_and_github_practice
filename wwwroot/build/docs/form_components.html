Form Wrapper

Under Construction

The code below is under revision to make the form sections more modular. This is pending review and will probably be implemented when we move forward with a full rebuild of the site.

The wrapper itself is created with a pair of block-level elements with classes of 'm-form-wrap' and 'form-interior' to create the proper layout. Additional form styles are contained within the form itself, so long as it's given a class of 'form'.

<div class="m-form-wrap">
    <div class="form-interior">
        <form action="#" method="post" accept-charset="utf-8" name="Example_form" class="form">
            <div class="form-item">
                <label for="example" class="left_label">Example:</label>
                <input type="text" name="example" placeholder="This is a basic example" />
                <div class="errormsg"><label>This is the error message. It will appear when something goes wrong.</label></div>
            </div>
        </form>
    </div>
</div>
