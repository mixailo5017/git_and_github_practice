Buttons

Buttons come in three types, off-white/gray, blue/white, and clear/white with the option to include an icon. Icons do not work on input buttons. The linkedin icon on the signup form is a sprite.

The class of 'std' was used to match the width of a general button in the comp by assigning a fixed with of 165 pixels. Use this if you have a button with short text and want to maintain uniform spacing.
